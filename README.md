# Life Console

> Customizable templates to fit your many documentation or tracking needs in life

Personal full-stack project built with React (Vite) + Typescript frontend and Node.js/Express + Typescript backend.

## Features (All tentative and subject to change)

- Customizable KanBan style board
  - Drag-and-Droppable card entries
  - Search & Filter functionality
- Customizable daily task tracker
  - Visual streaks as an emotional incentive
  - Multiple tracking capabilities
- ??

## Tech Stack

- Frontend: React 19 + Typescript + Vite + Tailwind CSS
- Backend: Node.js + Express + tsx

## Motivation and Thoughts

When I entered my final semester in university, pressure to start job hunting and the typical "Computer Science (CS) student imposter syndrome" led to an idea that would address both simulataneously — a complete full-stack application that helps with refreshing Web Development and Software Engineering competencies, but also doubles up as a way to document and organize the job search process.

After some thought and discussion with a friend, the idea to extend the application customisability became an interesting point. The original jobs board was planned to allow for custom named columns to accomodate application processes which may have less conventional phases, essentially allowing for more specificity.

Now, what if we lift the restriction on the "job entry" cards. Well, we essentially reduce the application to be some sort of Trello or Notion imitation. While I am not fond of projects that are simply clones of an existing product, I believe one that involves this customization feature definitely complicates things. The long term goal is to have multiple of such templates available to accomodate any thing I may need to organize in daily life whether it be fitness, habits etc. However, the plan is to start by implementing one template first and then work from there.
